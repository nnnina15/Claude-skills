# 色彩理論進階指南

## 色彩模型

### HEX / RGB（數位用）
- HEX：網頁、UI 設計標準
- RGB：螢幕顯示用，R(0-255) G(0-255) B(0-255)

### CMYK（印刷用）
- C=Cyan, M=Magenta, Y=Yellow, K=Black
- 數位轉印刷時注意色差，建議以 Pantone 對色

### Pantone（品牌標準色）
- 跨廠商、跨材質的色彩標準
- 品牌手冊必須標注 Pantone 色號

---

## 常用配色方案

### 單色配色（Monochromatic）
同一色相的深淺變化，視覺和諧，適合極簡品牌

### 互補色（Complementary）
色環對面的顏色（如藍-橙），強烈對比，高記憶度

### 類比色（Analogous）
色環相鄰顏色（如藍-靛-紫），協調自然

### 三角配色（Triadic）
色環等距三色，平衡且有活力

## 品牌色彩對比度規範

根據 WCAG 2.1 無障礙標準：
- 正文文字對比度 ≥ 4.5:1
- 大標題文字對比度 ≥ 3:1
- 使用工具：contrast-ratio.com

## 東亞市場色彩文化參考

| 顏色 | 台灣/華人市場意涵 | 注意事項 |
|------|-----------------|---------|
| 紅色 | 喜慶、吉祥、熱情 | 避免用於葬儀相關 |
| 白色 | 純潔（年輕世代）/ 哀悼（傳統） | 視受眾年齡層調整 |
| 金色 | 富貴、高端、尊榮 | 奢侈品牌常用 |
| 綠色 | 自然、健康、環保 | 科技業避免太深的綠（像素格感） |
| 藍色 | 科技、信任、專業 | 台灣科技業主色，需強化差異化 |
