## 字型層級規範

### 數位設計（螢幕用）
```
H1 標題：48-64px / Bold / 行高 1.2
H2 副標：32-40px / SemiBold / 行高 1.3
H3 段落標題：24-28px / Medium / 行高 1.4
Body 內文：16-18px / Regular / 行高 1.6
Caption 說明：12-14px / Regular / 行高 1.5
```

### 印刷設計（紙本用）
```
封面標題：36-72pt / Bold
內頁標題：18-24pt / SemiBold
小標：14-16pt / Medium
內文：10-12pt / Regular / 行高 15-18pt
頁尾：8-9pt / Light
```

---

## 排版黃金法則

1. **字型種類不超過 3 種**：標題、副標、內文各一，超過會顯得雜亂
2. **字重對比創造層次**：重/輕的搭配比單一字重更有設計感
3. **行距是內文易讀性關鍵**：中文行高建議 1.5-1.8 倍
4. **字距微調**：標題可拉開字距（tracking）增加質感
5. **避免全大寫中文**：中文沒有大小寫，全大寫只適用英文標題
