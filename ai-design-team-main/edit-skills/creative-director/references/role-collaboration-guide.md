# 角色協作流程詳解

## 標準影片製作流程（完整版）

```
任務輸入
    ↓
creative-director（任務分析與分派）
    ↓
[業配案] brief-analyst → 解析需求
[AI工具] ai-researcher → 研究工具
    ↓
content-planner → 企劃主題與結構
    ↓
creative-director（審核企劃）
    ↓
script-writer → 撰寫腳本
    ↓
creative-director（審核腳本）
    ↓
video-director → 規劃分鏡
brand-design/marketing-materials → 視覺素材
    ↓
project-manager → 整理交付清單 + 時程
    ↓
social-media → 社群配套貼文
    ↓
輸出完成
```

## 快速任務流程（輕量版）

適用於不需要全員參與的小任務：

```
單一貼文：content-planner → script-writer → social-media
素材設計：brand-design → marketing-materials
工具研究：ai-researcher → content-planner
業配解析：brief-analyst → creative-director（建議方向）
```

## 角色溝通原則

每個角色的輸出都要包含「給下一個角色的交接說明」：
- 我做了什麼
- 我的假設是什麼
- 下一個角色需要注意什麼
- 哪些地方需要確認

## 資訊不足時的處理原則

任何角色在資訊不足時，不要停下來什麼都不做，而是：
1. 列出已知條件
2. 列出缺少的條件
3. 說明缺少的條件如何影響產出
4. 基於最合理的假設先行推進
5. 標注「[待確認]」的部分

這樣確保整個流程不會卡住。
