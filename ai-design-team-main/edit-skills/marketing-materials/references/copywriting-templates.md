# 行銷文案模板庫

## 海報/廣告標題公式

### 公式 1：數字 + 好處
```
「3 步驟讓你的品牌被記住」
「7 天學會 XX，從零開始也能做到」
「省下 50% 時間，完成更多事」
```

### 公式 2：問句引發共鳴
```
「還在為 [痛點] 煩惱嗎？」
「你知道 [驚訝事實] 嗎？」
「[目標受眾]，這是你一直在找的解法」
```

### 公式 3：對比轉變
```
「從 [現況] 到 [理想]，只需要 [時間/方式]」
「告別 [舊方式]，迎接 [新方式]」
```

### 公式 4：社群證明
```
「超過 [數字] 人選擇的 [產品/服務]」
「[客戶名稱] 都在用的 [解決方案]」
```

## CTA（行動呼籲）文案庫

### 緊迫型
- 「立即搶購，限量 [X] 件」
- 「今天截止，把握最後機會」
- 「倒數 [X] 天，錯過等明年」

### 利益型
- 「免費試用 [X] 天」
- 「立即省 [X]%」
- 「免運費，今天下單」

### 好奇型
- 「點擊看更多」
- 「解鎖完整內容」
- 「看看他們怎麼說」

### 輕鬆型
- 「了解更多」
- 「探索我們的服務」
- 「預約免費諮詢」

## 各類素材文案結構

### 促銷活動素材
```
[折扣幅度/好處] — 最大最顯眼
[活動名稱] — 次要
[有效期間] — 清楚標示
[適用條件] — 最小字
[CTA] — 按鈕樣式
[品牌 Logo]
```

### 新品上市素材
```
[「全新」「終於來了」等引爆點] — 先聲奪人
[產品名稱/系列名稱] — 主標
[核心賣點 1-2 點] — 重要資訊
[上市日期] — 建立期待
[CTA：預購/了解更多]
[品牌 Logo]
```

### 活動邀請素材
```
[活動名稱] — 主標（大字）
[活動日期+時間] — 清楚，易讀
[活動地點/形式（線上/實體）]
[主辦單位/講師]
[費用（免費/付費）]
[CTA：立即報名]
[品牌 Logo]
```

### 節慶賀圖
```
[節日問候語] — 溫暖、感性
[品牌想傳遞的情感/價值觀] — 1-2 句
[感謝語或祝福語]
[品牌 Logo]（低調）
```

## 台灣在地化文案技巧

1. **用台灣習慣用語**：「揪」（揪朋友）、「衝」（衝動購買）、「爆滿」
2. **節日文化融入**：中秋月餅、尾牙、春酒等本土節慶
3. **網路用語（視品牌調性）**：「gg」「神操作」「懶人包」「整理一下」
4. **數字文化**：避免 4（諧音「死」），善用 6（順）、8（發）、9（久）
