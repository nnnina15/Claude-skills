# 各平台演算法解析（2025 版）

## Instagram 演算法

### 核心信號（重要性排序）
1. **互動歷史**：你與哪些帳號有過互動
2. **內容相關性**：AI 判斷內容是否符合用戶興趣
3. **發文時效性**：新貼文優先
4. **帳號活躍度**：定期發文的帳號有加分

### 提升觸及率的策略
- 發文後前 60 分鐘積極回覆留言（觸發演算法判定「熱門內容」）
- 用 Stories 互動功能（Poll、Question、Quiz）
- Reels 完播率 > 靜態圖片觸及率
- 跨格式互動（Reel + 貼文 + Stories 三管齊下）

### 2025 重點變化
- 原創內容加權，轉發/截圖貼文降權
- 短影音（Reels < 90秒）獲得最高推薦機率
- 帳號「利基度」重要（全面性帳號不如垂直主題帳號）

## Facebook 演算法

### 核心信號
1. **互動可能性**：預測你會不會對這則貼文有反應
2. **內容類型偏好**：依用戶歷史行為推斷
3. **朋友和家人互動**：被朋友互動過的內容優先
4. **可信度**：假新聞/農場文降權

### 提升觸及率的策略
- 觸發有意義的互動（留言 > 按讚）
- 避免「互動誘餌」（例：「按讚拿好運」→ 平台主動降權）
- 原生影片比 YouTube 連結觸及率高 2-4 倍
- 貼文後 3 小時內互動率決定後續觸及

## YouTube 演算法

### 核心指標
1. **點擊率（CTR）**：縮圖+標題決定是否被點擊（目標 > 5%）
2. **觀看時長（Watch Time）**：絕對時長，不是百分比
3. **觀眾滿意度**：按讚、不按讚、留言、分享
4. **完播率**：如何讓觀眾看完

### 提升演算法的策略
- 標題前 60 字含關鍵字（SEO 重要）
- 縮圖：大字 + 人臉表情 + 對比色
- 前 30 秒不要放片頭，直接進主題
- 在影片中鼓勵留言（提問）比呼籲按讚更有效

## TikTok 演算法

### 核心機制（漏斗式推送）
1. 新影片先推給 200-300 人小測試池
2. 完播率 > 70% → 推給更大群體
3. 層層放大，直到觸及率停滯

### 關鍵指標（重要性排序）
1. **完播率**（最重要）
2. **重播次數**
3. **分享數**
4. **留言數**
5. **按讚數**（反而最不重要）

### 策略
- 前 3 秒必須抓眼球，不然直接滑走
- 循環播放設計：結尾銜接開頭
- 用文字字幕：很多人靜音看
- 熱門音樂加權 + Hashtag Challenge 效果好
