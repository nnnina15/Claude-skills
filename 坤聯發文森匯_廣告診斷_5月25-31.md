## 二、Meta Ads 診斷

### ▌名單活動（名單25/9/17）— ⚠️ 問題最大

**7天關鍵指標：**
| 指標 | 數值 |
|------|------|
| 總花費 | 6,765 NTD |
| 總 Leads（表單/訊息） | 13 筆 |
| 平均 CPL | **~520 NTD／筆** |
| 平均 CPM | ~200–350 NTD |
| 平均 CTR | 1.0–3.4% |

**各 AdSet 表現：**

| AdSet | 7天花費 | Leads | CPL |
|-------|---------|-------|-----|
| 潛力族群 | 2,405 | 7 | ~344 NTD |
| 房產消費力 | 1,962 | 3 | ~654 NTD |
| 廣投 | 1,900 | 4 | ~475 NTD |
| 精密科學園區 | 76 | 0 | — |

**問題診斷：**

1. **精密科學園區 5/27 後幾乎停投（當日僅 $5）**，疑似預算耗盡或人工暫停，應確認原因。若因為效果差停投，代表受眾定義需調整。

2. **CPL 波動極大**：  
   - 潛力族群 5/26 CPL=456 NTD、5/27 CPL=334 NTD、5/28 CPL=78 NTD  
   → 單日 CPL 從 78 到 456 差距 6 倍，顯示受眾飽和或競價不穩定，建議設定出價上限或改成最低成本出價策略。

3. **房產消費力 5/27 單日花費 479 NTD，僅 1 筆 lead (CPL=479)**  
   → 該受眾 CTR=1.49%、CPC=19.96 — 受眾質量偏低，建議縮小或更換受眾條件。

4. **名單活動佔總預算 55%，但 CPL 500+ NTD 偏高**（房產案 CPL 目標建議壓在 300 NTD 以下）。整體投報率需重新評估。

5. **purchase_roas 全部為 null** — 沒有串接電商轉換事件，無法評估真實 ROI，建議確認 Pixel 事件設定。

3. 流量活動未設定轉換事件，無法判斷實際引流效果，建議加上 ViewContent 或自訂事件。

## 三、Google Ads 診斷

### ▌關鍵字活動（文森匯_關鍵字26/4/9）— ⚠️ 轉換追蹤設定異常

**7天關鍵指標：**
| 指標 | 數值 |
|------|------|
| 總花費 | ~2,021 NTD |
| 總點擊 | ~147 次 |
| 平均 CPC | 9.3–17.7 NTD |
| 顯示 Conversions | ~269 筆 |
| Conversion Value | 極低（每筆 ≈ 0.5） |
| ROAS | 0 或 null |

**問題診斷：**

1. **⚠️ Conversion 追蹤嚴重異常**：7天顯示 269 筆轉換，但每筆 conversion_value 僅約 0.5，且 ROAS 顯示為 0。  
   → **這些幾乎肯定是微轉換（如頁面停留、捲動、影片觀看），而非真實名單或成交。** 轉換設定需立即重新檢視，否則 Google 的出價演算法會以錯誤信號優化，導致廣告費浪費。

2. **5/27 單日花費 599 NTD，是其他日的 2–3 倍**（其他日平均 ~230 NTD）。  
   → 需確認是否有手動提高出價或 budget 設定異常，建議設定每日預算上限。

3. **CPC 9–18 NTD** 對搜尋廣告屬正常水準，關鍵字選詞方向應確認是否聚焦在有購買意圖的關鍵詞。
